# 8chip

8chip (read: ha-chip) is an experimental chip-8 emulator

## Requirements
- Rust 1.30+
- libsdl2-dev

## Usage
Compile by run
```shell script
cargo build --release
```
and then run the binary on target
```shell script
./target/release/hachip <rom path> (Unix)
or
target\release\hachip <rom path> (Windows)
```
